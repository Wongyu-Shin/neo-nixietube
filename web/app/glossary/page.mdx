# 용어 사전 (Glossary)

## ⚡ 가스 방전 — 닉시관의 원리

<GlowDischargeDiagram />

<span id="glow" />

### 글로우 방전 (Glow Discharge)
저압 가스에 전압을 인가하면 가스 원자가 이온화되면서 발생하는 자체 발광 현상. 닉시관의 핵심 원리. 형광등, 네온사인도 동일 원리.

- **음극 글로우(Cathode Glow):** 음극(숫자 형상 전극) 표면에 형성되는 발광 영역. 이것이 "숫자가 빛나는" 시각적 효과의 원인.
- **양광주(Positive Column):** 양극과 음극 사이의 긴 발광 영역. 네온사인의 빛은 이것.
- **음극 암부(Cathode Dark Space / Crookes Dark Space):** 음극 바로 앞의 어두운 영역. 여기서 전압 강하의 대부분이 발생.

<span id="paschen" />

### 파셴 법칙 (Paschen's Law)
항복 전압 V_b가 가스 압력(p)과 전극 간격(d)의 곱(p·d)에 의존한다는 법칙.
- `V_b = B × p × d / (ln(A × p × d) - ln(ln(1 + 1/γ)))`
- A, B: 가스별 상수 | γ: 2차 전자 방출 계수

**프로젝트에서의 의미:** p·d를 조절하여 원하는 동작 전압을 설정. 시뮬레이터(MAPE 1.79%)로 최적 조건 사전 결정.

<GlowStructure />

<TownsendCascade />

<span id="townsend" />

### 타운센드 메커니즘 (Townsend Mechanism)
전자가 가스 원자와 충돌하여 이온화(전자사태, electron avalanche)하는 과정. 방전 시작의 핵심. 1개 전자 → 2 → 4 → 8 → ... 기하급수적 증가. 이온이 음극을 때려 2차 전자를 방출(γ)하면 자속(self-sustaining) 방전으로 전환.

<span id="sputtering" />

### 스퍼터링 (Sputtering)
이온이 음극 표면에 충돌하여 금속 원자를 튕겨내는 현상. 유리 내벽이 흑화(blackening)되는 원인. 닉시관 수명의 주요 제한 요인.
- **완화 방법:** MgO 졸-겔 코팅(Bridge 12), 전해 니켈(99.5%+) 사용, 전류 제한(2mA 이하)

<span id="vb" />

### 항복 전압 (Breakdown Voltage, V_b)
가스에 전압을 인가했을 때 방전이 시작되는 최소 전압. 파셴 법칙으로 예측.
- **전통 닉시관:** 170-200V DC (Ne, 15 Torr, 1.5mm gap)
- **Penning 혼합 + MgO:** 120-140V DC (추정, 30-50% 저감)

<span id="penning" />

### Penning 혼합가스 (Penning Mixture)
Ne 98-99.5% + Ar 0.5-2% 혼합. Ne의 준안정 상태(16.62 eV)가 Ar(이온화 15.76 eV)를 이온화 → 항복 전압 10-30% 감소. 120V 네온 램프의 표준 충전 가스. MgO 코팅과 결합 시 30-50% 전압 저감 시너지. **추가 비용 ₩0.**

## 🔒 봉착 기술 — 관을 밀봉하는 방법

<SealingMethodsComparison />

<span id="gtm" />

### 유리-금속 봉착 (Glass-to-Metal Seal)
유리를 녹여 금속 와이어/핀 주변에 기밀(hermetic) 접합하는 기법. 닉시관의 최대 병목.
- **열팽창계수(CTE):** 유리와 금속의 CTE가 일치해야 냉각 시 크랙 방지.
  - 소다라임 유리: ~9.0×10⁻⁶/K | 듀멧 와이어: ~9.0×10⁻⁶/K (매칭)
  - 보로실리케이트: ~3.3×10⁻⁶/K | 코바르: ~5.0×10⁻⁶/K (매칭)

<CTEChart />

<span id="frit" />

### 프릿 실링 (Frit Sealing)
저융점 유리 분말(frit)을 사용하여 450°C에서 유리-유리 또는 유리-금속을 접합하는 기법. PDP 산업에서 발전.
- **Bi₂O₃ 계 프릿:** 무연(lead-free), 융점 350-500°C
- **Embedded lead seal:** 전극 리드선을 프릿 접합면에 매립하여 동시 봉착

<span id="butyl" />

### 부틸 고무 (Butyl Rubber / IIR)
모든 고무 중 가스 투과율이 가장 낮은 합성 고무. 건축 복층유리(IGU)와 자동차 전조등에서 40년간 사용.
- **핫멜트 부틸:** 200°C에서 유동, 상온에서 기밀 고체
- **네온 투과율:** ~10⁻¹⁰ cm³·cm/(cm²·s·cmHg) → PoC 수명 ~100일

<span id="solgel" />

### 졸-겔 (Sol-Gel)
금속 알콕사이드를 가수분해→축합하여 유리질 박막을 형성하는 화학 기법. "유리를 녹이지 않고 화학적으로 성장."
- **SiO₂ 졸-겔:** TEOS 기반. 기밀 보강 + 스퍼터링 방호.
- **MgO 졸-겔 (Bridge 12):** 마그네슘 아세테이트 기반. **방전 전압 저감** (γ ≈ 0.5) + 스퍼터링 방호. PDP 20년 검증. 400°C 어닐링.

<span id="dumet" />

### 듀멧 와이어 (Dumet Wire)
Cu 코어 + Fe-Ni 외피. CTE가 소다라임 유리와 일치하여 유리-금속 봉착에 최적. 프릿에 매립하여 기밀 관통 형성. ∅0.5mm, ₩20K.

<span id="torrseal" />

### Torr Seal (UHV 에폭시)
Varian/Agilent 9530001. 투과도 ~1×10⁻¹⁵의 UHV급 에폭시. 부틸 내층 위 **외층 가스 차단**으로 사용. 24시간 실온 경화(또는 65°C 2h 가속). &lt;10⁻⁹ Torr·L/s 누출율. ₩35K.

<span id="pib" />

### PIB (폴리이소부틸렌)
부틸의 순수 형태. 가스 투과율 부틸의 1/10 → 수명 100일 → 1,000일+. 태양전지 모듈 밀봉에 사용 (Bridge G4). ₩15K.

<span id="preform" />

### 유리 솔더 프리폼 (Glass Solder Preform)
링/디스크 형태의 성형 저융점 유리. 320-440°C에서 기밀 접합. 프릿 분말 자체 조합을 **기성품 구매**로 대체 (Bridge 15). NAMICS, OZ Optics 공급.

## 8 전극 — 숫자가 빛나는 부품

<HeaderAssemblyDiagram />

<span id="nickel" />

### 전해 니켈 (Electrolytic Nickel)
99.5%+ 고순도 니켈. 불순물(황, 탄소)이 적어 스퍼터링 속도 낮음. 0.2mm 포일에서 숫자 형상을 커팅/에칭하여 음극 제작. ₩20K.

<span id="aao" />

### AAO (양극산화알루미늄)
Al 양극산화 → 나노미터 규칙 다공 구조 자가조립 → Ni 전해도금 → **나노팁 어레이** → 전계 집중으로 글로우 균일성 개선 (Bridge 2).

<span id="plating" />

### 전해도금 / 전해도형 (Electroplating / Electroforming)
전해 용액에서 금속 이온을 환원시켜 표면에 금속막 형성. AAO 나노구조 Ni 충전, 3D 프린트 전극 셸 형성 (Bridge 13)에 사용. Watts bath(NiSO₄+NiCl₂+H₃BO₃) 표준.

<span id="mica" />

### 마이카 (Mica)
천연 운모. 전기 절연 + 내열. 전극 간 스페이서로 0.5-1mm 간격 유지 + 크로스토크 방전 방지.

<span id="etch" />

### FeCl₃ 에칭 (Ferric Chloride Etching)
염화제이철 40%에 금속 침지 → 화학 식각. 마스킹(네일 폴리시/토너) → 에칭(10-20분) → 세척. 레이저 커터 대안. 자택 가능. ₩5K.

## ◎ 진공/가스 — 관 내부 환경

<VacuumScaleDiagram />

<span id="pump" />

### 진공 펌프 (Vacuum Pump)
- **로터리 펌프:** ~10⁻³ Torr. 닉시관 "거친 배기"용. 네온사인 업체/대학에서 접근.
- **확산/터보 펌프:** ~10⁻⁶ Torr. 고진공용. PoC에는 불필요.

<span id="pirani" />

### 피라니 게이지 (Pirani Gauge)
열전도율 기반 진공도 측정기. 필라멘트 저항 변화 → 압력 측정. **DIY 버전:** 소형 전구 + ADS1115 + Arduino = ₩8.5K (상용 ₩50-100K 대비 95% 절감). 미니가마 Arduino와 공유.

<span id="map" />

### MAP (Modified Atmosphere Packaging)
식품 산업의 가스 치환 포장. 진공 없이 가스 조성 제어. 네온을 관류 플러싱하여 공기 치환 + O₂ 센서 모니터링 (Bridge G3).

<span id="getter" />

### 게터 (Getter)
밀봉된 관 내부의 잔류 가스를 화학적으로 흡착하는 소재.
- **Ba 게터:** RF 유도가열로 플래싱. 은색 반사막.
- **Ti 와이어:** 저가 대안 (₩5K). 700°C+에서 O₂/N₂ 흡착. 반복 플래싱 가능.

<span id="tipoff" />

### 팁오프 (Tip-off)
배기관을 토치로 녹여 밀봉. 전통 제조의 최종 단계.

<span id="leak" />

### 리크 테스트 (Leak Test)
- **Hold test:** 펌프 → 10⁻² Torr → 차단 → 1h 후 ΔP < 1 Torr면 PASS
- **스파크 코일:** 외부 방전으로 리크 지점 탐지 (네온사인 업계 표준)
- **방전 색상:** 공기 유입 시 오렌지 → 보라/핑크 변색

## ⊕ 구동/안전 — 전기 회로

<span id="boost" />

### 부스트 컨버터 (Boost Converter)
DC 5-12V → 100-500V 승압. 닉시관 구동 필수. NCH8200HV 등 전용 모듈 ₩15K. Arduino SHDN 핀으로 on/off.

<span id="bleed" />

### 블리드 저항 (Bleed Resistor)
고전압 캐패시터 잔류 전하를 안전 방전. 전원 off 후 수초 내 200V → 0V. **비협상 안전 장비.**

<span id="to8" />

### TO-8 기밀 헤더 (Hermetic Header)
14핀 금도금 기밀 피드스루. 코바르 핀이 유리를 관통하여 기밀 봉착된 상태로 납품. "핀 관통 봉착" 병목을 **기성품 구매로 해결** (Bridge G5). ₩20-30K.

## 🔧 자택 장비 — Arduino 에코시스템

<MicrowaveKilnSetup />

<span id="sic" />

### SiC 서셉터 (Silicon Carbide Susceptor)
전자레인지 미니가마의 발열체. SiC가 마이크로파(2.45 GHz) 흡수 → 열 변환. 최대 ~900°C. ₩50K.

<span id="pid" />

### PID 제어 (PID Control)
비례-적분-미분 피드백 제어. Arduino + MAX6675 열전대 + SSR로 미니가마에 구현. 목표 온도 프로파일(350°C → 450°C) 자동 추적. Bang-bang보다 오버슈트 적음. ~50줄 코드.

<span id="ssr" />

### SSR (Solid State Relay)
반도체 무접점 릴레이. Arduino 3.3/5V → 220V AC 전자레인지 전원 on/off. 25A 정격, ₩8K.

<span id="anneal" />

### 어닐링 (Annealing)
유리/금속 서서히 냉각 → 내부 열응력 제거. 프릿 소성 후 2°C/min 이하 서냉. 미니가마 PID 냉각 프로파일로 제어.
