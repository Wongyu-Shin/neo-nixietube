## 핵심 원칙: "소유 vs 접근" 패러다임

이 프로젝트에 필요한 장비 중 상당수는 **이미 서울 어딘가에 존재한다.** 문제는 소유가 아니라 접근이다.

| 전략 | 비용 | 예시 |
|------|------|------|
| **접근 (₩0)** | 무료 | 대학 실험실, 네온사인 업체 협업 |
| **대여 (₩~)** | 시간당/회당 | 메이커스페이스 장비 (레이저 커터 등) |
| **구독 (₩/월)** | 월정액 | 메이커스페이스 멤버십 |
| **구매 (₩₩₩)** | 일회성 | 최후 수단 — 접근 경로가 모두 실패한 후 |

> **경로 2의 전략적 의미:** 경로 2(상온 봉착)는 단순히 "저렴한 대안"이 아니다. **장소와 장비가 확보되지 않은 현실에서 즉시 시작할 수 있는 유일한 경로**다. 경로 2로 PoC를 달성하면서 동시에 경로 1에 필요한 관계(네온사인 업체, 대학)를 구축하는 것이 최적 전략이다.

## 자택 작업 가능 범위

경로 2(상온 봉착)의 대부분은 자택에서 가능하다. 다만 조건이 있다:

| 작업 | 자택 가능? | 조건 |
|------|-----------|------|
| 전극 설계 (CAD) | **가능** | PC만 필요 |
| 니켈 포일 화학 에칭 | **가능** | FeCl₃, 환기 |
| 부틸 봉착 | **가능** | 히트건, 환기 |
| Torr Seal 도포 | **가능** | 환기 (에폭시 혼합) |
| 네온 가스 플러싱 | **주의** | 네온은 무독성이나 O₂ 치환 → 환기 필수 |
| 고전압 점등 테스트 | **가능** | 절연 장갑, one-hand rule |
| 납땜 | **가능** | 환기 |
| AAO 양극산화 | **불가** | 산성 용액 → 대학 화학 후드 필요 |
| 프릿 소성 | **불가** | 450°C → 전기로 필요 |
| 진공 배기 | **불가** | 로터리 펌프 필요 |
