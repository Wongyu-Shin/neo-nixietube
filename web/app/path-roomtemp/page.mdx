## Step-by-Step Tutorial

### Step 1: 전극 제작 (Day 1)

1. 니켈 포일에 숫자 "8" 패턴 프린트 (레이저 프린터로 마스크 출력)
2. 메이커스페이스 레이저 커터로 커팅 (또는 FeCl₃ 에칭)
3. 디버링: 미세 사포(#1000)로 에지 정리
4. IPA(이소프로필 알코올)로 세척 → 건조

### Step 2: 전극을 기밀 헤더에 장착 (Day 1)

1. TO-8 14핀 기밀 헤더의 핀 배치 확인
2. 각 숫자 전극의 리드선을 해당 핀에 스폿 용접 또는 납땜
   - 납땜 주의: 플럭스 잔류물이 방전 오염 가능 → 로진 플럭스 + IPA 세척
3. 양극 메쉬를 최전면 핀에 연결
4. 전극 간 절연 확인 (멀티미터로 핀 간 저항 측정 — MΩ 이상)

### Step 3: 유리 돔 조립 (Day 2)

1. 유리 돔/관을 TO-8 헤더 위에 올림 (전극이 내부에 위치)
2. 돔-헤더 접합면에 핫멜트 부틸 테이프를 한 바퀴 감기
3. 히트건(200°C)으로 부틸 연화 → 돔을 눌러 밀착
4. 냉각 후 폴리설파이드 실란트를 외부에서 2차 도포
5. 24시간 경화 대기

> **배기관:** 유리 돔 상단에 미리 가는 유리 튜브(OD 5mm)를 에폭시로 접합 — 이것이 가스 플러싱용 입구

### Step 4: 가스 플러싱 (Day 3)

MAP(가스 치환 포장) 원리를 적용:

1. 배기관에 실리콘 호스 연결 → 네온 가스 레귤레이터에 연결
2. 돔 하단(부틸 봉착부)에 배출구 역할의 가는 구멍 확보 (또는 2번째 튜브)
3. 네온 가스를 0.5 L/min으로 3-5분 플러싱
4. O₂ 센서로 배출구의 산소 농도 모니터링
5. **O₂ < 0.5% 확인** (3회 플러싱-퍼지 사이클)
6. 배출구를 에폭시로 밀봉
7. 배기관도 에폭시로 밀봉

> **팁:** 게터(활성탄 또는 Ba 게터)를 관 내부에 배치하면 잔류 O₂/H₂O를 추가 흡착 → 순도 향상

### Step 5: 점등 테스트 (Day 3)

1. 부스트 컨버터 출력을 **300-500V**로 설정 (고압 동작)
   - 정확한 전압은 파셴 시뮬레이터로 사전 계산
2. 양극 저항 15-33kΩ 연결
3. **절연 장갑 착용, one-hand rule**
4. 전원 ON
5. 네온 오렌지 글로우 확인!

### Step 6: 에이징 및 관찰 (Week 1-4)

1. 1mA에서 연속 점등 시작
2. 매일 사진 촬영 → 밝기, 색상, 흑화 진행 기록
3. O₂ 센서로 수일마다 잔존 산소 확인 (센서를 관 외벽에 테이프로 부착하여 누출 감지)
4. 100시간 연속 점등 달성 시 **PoC 성공**

## 예상 실패 모드와 대응

| 실패 | 원인 | 대응 |
|------|------|------|
| 점등 안 됨 | 가스 순도 부족 (잔류 공기) | 플러싱 횟수 증가 + 게터 추가 |
| 수일 후 어두워짐 | 부틸 봉착부 네온 투과 | 졸-겔 SiO₂ 오버코팅 추가 |
| 이상한 색상 (보라/분홍) | 공기 유입 | 봉착부 리크 탐색 → 재봉착 |
| 전극 형상 안 보임 | 고압 글로우 특성 (양광주 지배) | 전류 조절 또는 압력 변경 |
