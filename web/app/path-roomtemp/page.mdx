  전극 제작
</StepHeader>

니켈 포일에서 숫자 형상의 음극(cathode)을 만드는 공정. 이 전극이 네온 글로우로 빛나는 부분이다.

### 옵션 A: 메이커스페이스 레이저 커팅 (권장)

1. **숫자 "8" 패턴 CAD 설계** — 포일 두께 0.2mm에 맞는 커프(kerf) 보정. 숫자 획 폭 최소 0.8mm (0.2mm 포일의 4배). AutoCAD/Fusion360 또는 Inkscape SVG.
2. **세운메이커스큐브 레이저 커터에서 커팅** — CO₂ 레이저 파워 15-25W, 속도 300-500mm/min. 니켈 0.2mm는 파이버 레이저 권장이나 CO₂도 가능(반사율 주의). 3회 패스로 관통.
3. **디버링** — #1000 미세 사포로 에지 버(burr) 제거. 버가 남으면 방전 시 아킹 유발. 핀셋으로 확인.
4. **세척** — IPA(이소프로필 알코올)에 5분 초음파 세척 → 건조. 표면 유분/파티클이 글로우 불균일 원인.

### 옵션 B: FeCl₃ 화학 에칭 (자택)

1. **마스킹** — 레이저 프린터 토너를 니켈 포일에 전사(아이론 180°C, 2분). 또는 네일 폴리시로 직접 페인팅.
2. **에칭** — FeCl₃ 40% 용액(₩5K, 화학시약점), 40-50°C(±5°C, 핫플레이트 또는 중탕) 가열, 15-20분. 에칭 레이트 ~0.01mm/min at 40°C. 0.2mm 관통에 약 20분. 5분마다 용액을 가볍게 흔들어 반응물(FeCl₂) 제거 — 교반하지 않으면 에칭 속도 저하.
3. **마스크 제거** — 아세톤으로 토너/네일폴리시 제거 → IPA 세척 → 건조.

<StepHeader id="step-2" step={2} day="Day 1" duration="약 1시간" color="#D4A853">
  전극을 기밀 헤더에 장착
</StepHeader>

커팅된 숫자 전극을 기밀 헤더(12-pin 압축 헤더, 바디 OD 12.7mm, Kovar 핀 ∅0.46mm)의 핀에 연결하고, 마이카 스페이서로 간격을 유지하는 공정. 헤더와 엔드캡 사이는 Torr Seal(Varian 9530001, UHV급 에폭시, &lt;10⁻⁹ Torr·L/s)로 기밀 봉착한다.

<PinAssignment />

1. **핀 배치 확인** — TO-8 14핀 헤더의 각 핀에 번호를 유성펜으로 표기. 핀 0-9: 숫자 음극, 핀 10: 양극 메쉬, 핀 11: 도트(선택), 핀 12-13: 예비. 핀 배치도를 인쇄하여 작업대에 비치.
2. **전극 리드선 납땜** — 각 숫자 전극의 리드선(0.3mm Ni 와이어, 길이 15-20mm)을 해당 핀에 납땜. **플럭스: 로진(RMA) 사용** — 산성 플럭스는 잔류물이 방전 시 오염 방출. 납땜 온도 350°C, 접촉 시간 3초 이내(과열 방지). 납땜 후 IPA로 플럭스 완전 세척.
3. **마이카 스페이서 삽입** — 전극 간 0.5-1.0mm(±0.2mm) 간격 유지. 마이카 시트(두께 0.5mm, ₩5K)를 전극 크기에 맞게 가위로 커팅. 최전면에 양극 메쉬(Ni 메쉬 #100, 유효폭 13.1mm — 엔벨로프 내벽에 chord-clip). 전극 적층 순서: 양극 메쉬 → 마이카 → 숫자 0 → 마이카 → 숫자 1 → ... → 숫자 9.
4. **절연 확인** — 멀티미터 저항 모드로 모든 핀 쌍(총 91쌍) 측정. 각 쌍에서 **> 1 MΩ** 이어야 함. kΩ 이하 → 납땜 브릿지 또는 마이카 오프셋. 특히 인접 핀 쌍에 집중.

유리 돔을 엔드캡 위에 IIR butyl rubber 내층(4mm 경로)으로 1차 봉착하고, Torr Seal 외층(3mm 경로)으로 2차 가스 차단하는 복합 실링 공정. Al CTE(23.6) vs 보로실리케이트 CTE(3.3)의 7배 차이를 부틸이 흡수.

<ButylApplicationProcess />

1. **접합면 탈지** — 유리 돔 하단 림과 TO-8 헤더 플랜지를 IPA에 적신 무진 와이프로 닦기. 지문/먼지/유분은 부틸 접착력을 30-50% 저하시킴. 닦은 후 1분 건조 — IPA가 완전 증발한 뒤 다음 단계.
2. **부틸 테이프 적용** — 핫멜트 부틸 테이프(폭 4mm)를 헤더 플랜지 둘레(∅15.2mm × π ≈ 48mm)에 한 바퀴 감기. **테이프 양 끝을 정확히 맞대기 — 겹침 불가(겹침은 두께 불균일 → 기밀 약점), 간극도 불가(가스 경로 형성)**. 부틸 테이프가 플랜지 외벽에서 1mm 이상 돌출하지 않도록 정렬.
3. **히트건 가열 + 밀착** — 히트건(200°C 설정) 노즐을 부틸에서 3-5cm 거리에서 10-15초 균일 조사. 부틸이 연화되어 약간 광택이 나면 적정(과열 시 흘러내림). 유리 돔을 수직으로 올려놓고 양손으로 균일하게 3초간 눌러 밀착. **편심 주의** — 한쪽만 눌리면 기밀 불균일.
4. **냉각 확인** — 상온에서 5분 방치. 부틸이 고체화되면 돔을 가볍게 좌우로 흔들어 밀착 확인 — 흔들림 없어야 함.
5. **폴리설파이드 2차 도포** — 코킹건에 폴리설파이드 실란트 카트리지 장착. 부틸 봉착부 외부에 폭 3-4mm 연속 비드(bead)로 한 바퀴 도포. **끊김 없이 연속** — 끊김 부위가 구조적 약점. 24시간 경화 대기(20°C 이상 환경).

> **배기관:** 유리 돔 상단에 OD 5mm 유리 튜브를 미리 에폭시로 접합해두어야 한다. 이것이 가스 플러싱의 입구 역할.
>
> **sight glass:** 상단 엔드캡의 중앙 관찰창(∅16mm 보로실리케이트/사파이어 디스크)은 butyl + Torr Seal 복합 실링으로 봉착 — IIR butyl rubber 내층이 CTE 차이를 흡수하고, Torr Seal 외층이 가스 차단.

</StepHeader>

식품 산업의 MAP(Modified Atmosphere Packaging) 원리를 적용하여 관 내부의 공기를 **Ne + 0.5% Ar (Penning mixture)**로 치환한다. Ar 0.5% 첨가로 점화 전압이 순수 Ne의 250V 이상에서 ~170V로 낮아져 표준 K155ID1/74141 드라이버 IC 호환. **진공 펌프 불필요.**

1. **배기관에 네온 가스 레귤레이터 연결** — 실리콘 호스(내경 5mm)로 배기관과 레귤레이터 연결. 유량 0.5 L/min 설정. 네온은 공기보다 가벼우므로 위에서 아래로 유입이 효율적.
2. **배출구 확보** — 돔 하단(부틸 봉착부)에 가는 구멍(∅1-2mm) 또는 2번째 유리 튜브를 미리 확보. 공기가 빠져나가는 경로.
3. **O₂ 센서 배치** — 배출구에 전기화학식 O₂ 센서 모듈을 밀착 배치. Arduino에 연결하여 실시간 모니터링 (또는 센서 내장 디스플레이).
4. **1차 플러싱** — 네온 0.5 L/min으로 3분 연속 플러싱. 관 체적 ~5-10mL이므로 30초면 이론적 3회 치환이지만, 잔류 공기 포켓을 위해 3분.
5. **O₂ 확인** — 배출구 O₂ 농도 확인. 목표: < 0.5%. 미달 시 추가 2분 플러싱.
6. **반복** — 총 3회 플러싱-퍼지 사이클. 각 사이클 사이에 10초 정지 → 잔류 공기 확산 → 재플러싱.
7. **배출구 밀봉** — O₂ < 0.5% 확인 후 배출구를 에폭시로 즉시 밀봉. 5분 경화 에폭시 사용.
8. **fill stem 크림핑** — 하단 엔드캡의 1/16" (1.5875mm OD) 구리 캐필러리 튜브를 주얼리 크림프 공구로 압착 밀봉. 크림프 후 Torr Seal로 크림프 부위를 덮어 2차 기밀 확보. 배기관(상단)도 에폭시로 밀봉.

> **팁:** Ti 와이어 게터(₩5K)를 관 내부에 미리 배치하면 잔류 O₂/H₂O를 화학 흡착 → 가스 순도 추가 향상. 점등 후 전극 과열로 Ti가 700°C 도달 시 자동 활성화.

<GoNoGo criteria={[

  { check: "에폭시 밀봉 완전성", pass: "경화 후 가시적 간극 없음", fail: "기포/간극 → 에폭시 추가 도포" },

<SafetyWarning>**이 단계부터 300-500V DC 고전압.** 절연 장갑(Class 0, 500V+) 착용 필수. one-hand rule — 한 손만 사용하여 양손으로 회로 양단을 동시에 만지는 상황 방지. 블리드 저항(1MΩ)이 출력 캐패시터에 연결되어 있는지 반드시 확인. 전원 off 후 최소 5초 대기(RC 방전 시간상수 = 1MΩ × 1μF ≈ 1초, 5τ = 5초).</SafetyWarning>

<CircuitDiagram />

1. **부스트 컨버터 설정** — 입력: 12V DC 어댑터(2A+). 출력: **300V에서 시작** (경로 2는 상압~약감압이므로 전통 닉시관보다 높은 전압 필요). 파셴 시뮬레이터 사전 계산값 참조. 출력 캐패시터에 블리드 저항(1MΩ, 1/4W) 병렬 연결.
2. **양극 저항 연결** — 15-33kΩ, 2W 저항을 양극(+)과 부스트 컨버터 출력 사이에 **직렬** 연결. 이것이 전류를 2-5mA로 제한. 전류 계산: I = (V_supply - V_breakdown) / R. 예: (400V - 350V) / 15kΩ = 3.3mA.
3. **절연 장갑 착용** — Class 0 절연 장갑(500V+ 내전압). one-hand rule 숙지. 작업대 주변 도전체 제거.
4. **전원 ON** — 가변 출력 부스트 컨버터: 300V부터 시작 → 50V 단위로 증가. 각 전압에서 5초 관찰. 고정 출력: 바로 설정 전압 인가.
5. **글로우 확인** — 네온 오렌지-적색 글로우가 관찰되면 **성공!** 글로우가 숫자 형상을 따라 발광하는지 확인. 스마트폰으로 사진 촬영(플래시 off, ISO 고정).

<StepHeader id="step-6" step={6} day="Week 1-4" duration="100시간 연속" color="#C17B5E">
  에이징 및 관찰
</StepHeader>

글로우 안정화와 수명 데이터 수집을 위한 장기 점등 관찰.

<AgingSetup />

1. **1mA 연속 점등 시작** — 양극 저항을 조정하여 전류 1mA(±0.2mA) 설정. 멀티미터로 양극 저항 양단 전압을 측정하여 전류 확인(V/R = I). 안정적 글로우 유지 확인 후 스탠드에 고정.
2. **매일 사진 촬영** — 스마트폰을 고정 스탠드(삼각대 + 폰 마운트, ₩10K)에 장착. 동일 각도/거리(30cm)/조명(암실 또는 차광)에서 촬영. 카메라 설정 고정: 수동 모드, ISO 400, 셔터 1/60s, WB 고정. 밝기·색상·흑화(blackening) 진행을 날짜별로 정량 비교.
3. **주간 O₂ 확인** — O₂ 센서를 관 외벽에 테이프로 밀착 부착. 배경 O₂(20.9%)와 비교하여 누출 감지. 센서 수치가 배경보다 0.5% 이상 낮으면 관 내부 가스 누출 의심.
4. **체크포인트:**
   - **24시간** — 1차 사진 + O₂ 체크. 글로우 안정 여부 확인.
   - **48시간** — 2차 사진. 흑화 진행 초기 확인.
   - **100시간** — 최종 평가. 글로우 유지 + 전류 안정(±10%) + 흑화 면적 < 10% + 색상 오렌지 유지 → **PoC 성공.**
