<StepHeader id="step-1" step={1} day="Day 1" duration="약 2시간" color="#B8A9C9">
  전극 제작
</StepHeader>

니켈 포일에서 숫자 형상의 음극(cathode)을 만드는 공정. 이 전극이 네온 글로우로 빛나는 부분이다.

### 옵션 A: 메이커스페이스 레이저 커팅 (권장)

1. **숫자 "8" 패턴 CAD 설계** — 포일 두께 0.2mm에 맞는 커프(kerf) 보정. 숫자 획 폭 최소 0.8mm (0.2mm 포일의 4배). AutoCAD/Fusion360 또는 Inkscape SVG.
2. **세운메이커스큐브 레이저 커터에서 커팅** — CO₂ 레이저 파워 15-25W, 속도 300-500mm/min. 니켈 0.2mm는 파이버 레이저 권장이나 CO₂도 가능(반사율 주의). 3회 패스로 관통.
3. **디버링** — #1000 미세 사포로 에지 버(burr) 제거. 버가 남으면 방전 시 아킹 유발. 핀셋으로 확인.
4. **세척** — IPA(이소프로필 알코올)에 5분 초음파 세척 → 건조. 표면 유분/파티클이 글로우 불균일 원인.

### 옵션 B: FeCl₃ 화학 에칭 (자택)

1. **마스킹** — 레이저 프린터 토너를 니켈 포일에 전사(아이론 180°C, 2분). 또는 네일 폴리시로 직접 페인팅.
2. **에칭** — FeCl₃ 40% 용액(₩5K, 화학시약점), 40-50°C 가열, 10-20분. 에칭 레이트 ~0.01mm/min. 0.2mm 관통에 20분. 용액을 가끔 흔들어 반응물 제거.
3. **마스크 제거** — 아세톤으로 토너/네일폴리시 제거 → IPA 세척 → 건조.

<StepHeader id="step-2" step={2} day="Day 1" duration="약 1시간" color="#D4A853">
  전극을 기밀 헤더에 장착
</StepHeader>

커팅된 숫자 전극을 TO-8 기밀 헤더의 핀에 연결하고, 마이카 스페이서로 간격을 유지하는 공정.

1. **핀 배치 확인** — TO-8 14핀 헤더의 각 핀에 번호 할당. 핀 0-9: 숫자 음극, 핀 10: 양극 메쉬, 핀 11: 도트(선택), 핀 12-13: 예비.
2. **전극 리드선 납땜** — 각 숫자 전극의 리드선(0.3mm Ni 와이어)을 해당 핀에 납땜. **플럭스: 로진(RMA) 사용** — 산성 플럭스는 잔류물이 방전 오염. 납땜 후 IPA로 플럭스 세척.
3. **마이카 스페이서 삽입** — 전극 간 0.5-1.0mm 간격 유지. 마이카 시트(₩5K)를 전극 크기에 맞게 커팅. 최전면에 양극 메쉬(Ni 메쉬 #100).
4. **절연 확인** — 멀티미터로 핀 간 저항 측정. 모든 핀 쌍에서 **MΩ 이상**이어야 함. kΩ 이하면 마이카 위치 재조정.

<StepHeader id="step-3" step={3} day="Day 2" duration="30분 작업 + 24시간 경화" color="#6BA368">
  유리 돔 부틸 봉착
</StepHeader>

유리 돔을 TO-8 헤더 위에 부틸 테이프로 1차 봉착하고, 폴리설파이드로 2차 보강하는 공정.

1. **접합면 준비** — 유리 돔 하단 림과 TO-8 헤더 플랜지를 IPA로 탈지. 먼지/유분은 부틸 접착력 저하 원인.
2. **부틸 테이프 적용** — 핫멜트 부틸 테이프(폭 4mm)를 헤더 플랜지 둘레에 한 바퀴 감기. **겹침 부위 없이 연속** — 겹침은 기밀 약점.
3. **히트건 가열 + 밀착** — 히트건 200°C를 부틸에 10-15초 조사 → 부틸이 연화(점착성 발현) → 유리 돔을 올려놓고 균일하게 눌러 밀착. **편심 주의** — 한쪽이 더 눌리면 기밀 불균일.
4. **냉각 확인** — 상온 냉각 5분. 부틸이 고체화되면 돔을 가볍게 흔들어 밀착 확인.
5. **폴리설파이드 2차 도포** — 부틸 봉착부 외부에 폴리설파이드 실란트를 연속 비드(bead)로 도포. **끊김 없이 한 바퀴 연속**. 24시간 경화 대기.

> **배기관:** 유리 돔 상단에 OD 5mm 유리 튜브를 미리 에폭시로 접합해두어야 한다. 이것이 가스 플러싱의 입구 역할.

<StepHeader id="step-4" step={4} day="Day 3" duration="약 15분" color="#7B9EB8">
  가스 플러싱 (MAP 방식)
</StepHeader>

식품 산업의 MAP(Modified Atmosphere Packaging) 원리를 적용하여 관 내부의 공기를 네온 가스로 치환한다. **진공 펌프 불필요.**

1. **배기관에 네온 가스 레귤레이터 연결** — 실리콘 호스(내경 5mm)로 배기관과 레귤레이터 연결. 유량 0.5 L/min 설정. 네온은 공기보다 가벼우므로 위에서 아래로 유입이 효율적.
2. **배출구 확보** — 돔 하단(부틸 봉착부)에 가는 구멍(∅1-2mm) 또는 2번째 유리 튜브를 미리 확보. 공기가 빠져나가는 경로.
3. **O₂ 센서 배치** — 배출구에 전기화학식 O₂ 센서 모듈을 밀착 배치. Arduino에 연결하여 실시간 모니터링 (또는 센서 내장 디스플레이).
4. **1차 플러싱** — 네온 0.5 L/min으로 3분 연속 플러싱. 관 체적 ~5-10mL이므로 30초면 이론적 3회 치환이지만, 잔류 공기 포켓을 위해 3분.
5. **O₂ 확인** — 배출구 O₂ 농도 확인. 목표: < 0.5%. 미달 시 추가 2분 플러싱.
6. **반복** — 총 3회 플러싱-퍼지 사이클. 각 사이클 사이에 10초 정지 → 잔류 공기 확산 → 재플러싱.
7. **배출구 밀봉** — O₂ < 0.5% 확인 후 배출구를 에폭시로 즉시 밀봉. 5분 경화 에폭시 사용.
8. **배기관 밀봉** — 네온 가스 라인 분리 → 배기관도 에폭시 밀봉.

> **팁:** Ti 와이어 게터(₩5K)를 관 내부에 미리 배치하면 잔류 O₂/H₂O를 화학 흡착 → 가스 순도 추가 향상. 점등 후 전극 과열로 Ti가 700°C 도달 시 자동 활성화.

<GoNoGo criteria={[
  { check: "O₂ 농도", pass: "< 0.5% (3회 사이클 후)", fail: "> 0.5% → 추가 플러싱 또는 리크 확인" },
  { check: "에폭시 밀봉 완전성", pass: "경화 후 가시적 간극 없음", fail: "기포/간극 → 에폭시 추가 도포" },
]} />

<SafetyWarning>네온 가스 자체는 무독성이나, 밀폐 공간에서 대량 방출 시 O₂ 치환(질식 위험). 반드시 환기된 공간에서 작업. 소형 실린더(1L)는 이 위험이 극히 낮으나 습관적 환기 권장.</SafetyWarning>

<SafetyWarning>**이 단계부터 170-200V DC(또는 Penning 가스 사용 시 120-140V DC) 고전압.** 절연 장갑 착용 필수. one-hand rule(한 손만 사용). 블리드 저항 확인. 전원 off 후 5초 대기(캐패시터 방전).</SafetyWarning>

1. **부스트 컨버터 설정** — 입력 12V DC (어댑터), 출력 **300-500V로 시작** (고압 동작 — 경로 2는 상압이므로 전통보다 높은 전압 필요). 파셴 시뮬레이터로 사전 계산된 최적 전압 참조.
2. **양극 저항 연결** — 15-33kΩ, 2W를 양극과 부스트 컨버터 사이에 직렬. 이것이 전류를 2-5mA로 제한. **블리드 저항**(1MΩ)을 출력 캐패시터에 병렬 — 전원 off 시 자동 방전.
3. **절연 장갑 착용, one-hand rule**
4. **전원 ON** — 전압을 천천히 올림 (가변 출력 시). 300V부터 시작 → 50V 단위 증가.
5. **글로우 확인** — 네온 오렌지 글로우가 관찰되면 **성공!** 글로우가 숫자 형상을 따라 발광하는지 확인.

글로우 안정화와 수명 데이터 수집을 위한 장기 점등 관찰.

1. **1mA 연속 점등 시작** — 양극 저항 조정으로 전류 1mA 설정. 안정적 글로우 유지 확인.
2. **매일 사진 촬영** — 동일 각도/조명에서 촬영. 밝기, 색상, 흑화(blackening) 진행 정량화. 스마트폰 카메라 + 고정 스탠드.
3. **주간 O₂ 확인** — O₂ 센서를 관 외벽에 테이프로 부착하여 누출 감지. 배경 O₂ 레벨과 비교.
