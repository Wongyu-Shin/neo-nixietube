### Bridge 3 가시성 분석 — 핵심 결과

> **질문:** 마이크로 격벽(500μm-1mm, 200+ Torr)에서 글로우가 전극 형상을 감싸는가?

| 조건 | dc (μm) | 글로우 (μm) | 글로우/갭 비 | 강도 (상대) | 판정 |
|------|---------|-----------|------------|-----------|------|
| **전통 (15 Torr, 1.5mm)** | **291** | **204** | **0.33** | **1x** | **기준** |
| 1mm @ 100 Torr | 67 | 47 | 0.11 | 40x | 얇지만 밝음 |
| 500μm @ 200 Torr | 45 | 32 | 0.15 | 180x | 얇지만 매우 밝음 |
| 1mm @ 300 Torr | 37 | 26 | 0.06 | 440x | 너무 얇음 |
| 500μm @ 500 Torr | 30 | 21 | 0.10 | 1300x | 극도로 밝은 얇은 선 |

### 해석

- **글로우/갭 비 0.15 이상:** 전통 닉시관의 약 절반 수준. 숫자 형상이 "윤곽선"으로 보임.
- **발광 강도 40-180x:** 육안 가시성은 충분. 어두운 환경에서 매우 선명.
- **판정:** "얇지만 매우 밝은 윤곽선 발광" — 전통 닉시관과 다른 미학이나, 독특한 시각적 효과.
- **최적 조건:** Ne, 100-200 Torr, 500μm-1mm gap

## 실행 방법

```bash
# 시뮬레이터 실행
cd sim
python3 paschen.py          # 파셴 계산기
python3 optimize.py         # 3 bridge 최적 조건
python3 glow_model.py       # 글로우 가시성 분석
python3 validate.py         # 파셴 LOOCV 검증
python3 validate_glow.py    # 글로우 모델 검증
python3 plot.py             # 파셴 곡선 시각화
```

Requirements: Python 3.10+, numpy, scipy, matplotlib
