## Week 2-3: 핵심 실험

### 실험 A: 부틸 봉착 기밀 테스트 (상온 경로)

**목표:** 부틸+폴리설파이드로 유리관을 봉착했을 때 기밀이 유지되는가?

1. 유리관 양끝에 유리 디스크를 부틸 테이프로 접합
2. 한쪽에 배기관(가는 유리 튜브) 에폭시 접합
3. 주사기로 부분 배기 (가능한 만큼)
4. 24시간 후 압력 변화 측정 (간이 기압계 또는 풍선식)
5. **PASS:** 시각적 변화 없음 → 기밀 충분

### 실험 B: 프릿 실링 기밀 테스트 (프릿 경로)

**목표:** 자체 조합 프릿으로 유리-유리 봉착 + 듀멧 와이어 매립 기밀이 유지되는가?

1. 프릿 페이스트 조합 (Bi₂O₃:B₂O₃:ZnO = 70:20:10)
2. 유리 디스크에 도포 + 듀멧 와이어 배치
3. 2단계 소성 (350°C → 450°C)
4. Hold test: 로터리 펌프로 10⁻² Torr → 1시간 유지
5. **PASS:** 압력 변화 < 1 Torr

### 실험 C: 네온 플러싱 순도 테스트 (상온 경로)

**목표:** MAP 방식 플러싱으로 잔존 O₂ < 0.5% 달성 가능한가?

1. 유리관 양끝 개방 (한쪽 입구, 한쪽 출구)
2. 네온 가스 0.5 L/min으로 플러싱
3. 출구에 O₂ 센서 배치
4. 3회 플러싱-퍼지 사이클
5. **PASS:** O₂ < 0.5%

## Week 4-6: 통합 PoC

### 경로별 첫 점등 시도

**상온 경로 (경로 2):**
1. TO-8 헤더에 전극 장착

4. 170-200V 인가 → 글로우 확인

## 의사결정 트리

```
Week 2 결과:
├── 실험 A(부틸) 성공 + 실험 C(플러싱) 성공
│   → 상온 경로 우선 추진 → Week 4에서 첫 점등
│
├── 실험 A 성공 + 실험 C 실패
│   → 상온 봉착 + 전통 진공 충전 하이브리드
│
├── 실험 B(프릿) 성공
│   → 프릿 경로 추진 → Week 5-6에서 첫 점등
│
├── 실험 A, B 모두 실패
│   → 네온사인 기술자 협업으로 전환
│   → 또는 상용 기밀 헤더 + 에폭시 봉착 (최소 경로)
│
└── 네온 가스 확보 실패
    → 아르곤으로 대체 (색상 달라짐, 보라색)
    → 네온사인 업체 충전 서비스 재시도
```
