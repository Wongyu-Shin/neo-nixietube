## Week 2-3: 핵심 실험

### 실험 A: 부틸 봉착 기밀 테스트 (상온 경로) — 자택 가능

**목표:** 부틸+Torr Seal 복합 실링으로 유리관을 봉착했을 때 기밀이 유지되는가?

1. 유리관 양끝에 유리 디스크를 부틸 테이프로 접합
2. 한쪽에 배기관(가는 유리 튜브) 에폭시 접합
3. 주사기로 부분 배기 (가능한 만큼)
4. 24시간 후 압력 변화 측정 (간이 기압계 또는 풍선식)
5. **PASS:** 시각적 변화 없음 → 기밀 충분

### 실험 B: 프릿 실링 기밀 테스트 (프릿 경로) — 자택(미니가마) + 대학(펌프)

**필요 장비:** 전자레인지 미니가마+Arduino PID (자택), 로터리 펌프 (대학 or 네온사인 업체)

1. 프릿 페이스트 조합 (Bi₂O₃:B₂O₃:ZnO = 70:20:10)
2. 유리 디스크에 도포 + 듀멧 와이어 배치
3. **전자레인지 미니가마+Arduino PID**로 자택에서 2단계 소성 (350°C → 450°C)
4. Hold test: 로터리 펌프로 10⁻² Torr → 1시간 유지
5. **PASS:** 압력 변화 < 1 Torr

### 실험 C: 네온 플러싱 순도 테스트 (상온 경로) — 자택 가능

**목표:** MAP 방식 플러싱으로 잔존 O₂ < 0.5% 달성 가능한가?

1. 유리관 양끝 개방 (한쪽 입구, 한쪽 출구)
2. 네온 가스 0.5 L/min으로 플러싱
3. 출구에 O₂ 센서 배치
4. 3회 플러싱-퍼지 사이클
5. **PASS:** O₂ < 0.5%
