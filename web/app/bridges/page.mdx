## 병목 1: 유리-금속 실링

전통 닉시관은 토치로 유리를 800-1000°C에서 녹여 금속 핀 주변에 봉착한다. Dalibor Farny도 이 기술을 마스터하는 데 2년 이상 소요. 아마추어의 최대 장벽. 이것을 대체하는 3가지 브릿지:

<BridgeSection
  color="#D4A853"
  title="프릿 실링"

  cost="₩50-100K"

>

<WhyNeeded>전통 토치 봉착(800-1000°C)은 숙련도에 2년+가 필요하고, 초기 실패율이 30-50%에 달한다. 프릿 실링은 이 온도를 450°C로 55% 낮추고, 토치 대신 전기로(또는 미니가마)를 사용하여 **재현 가능한 온도 프로파일**로 봉착한다.</WhyNeeded>

프릿 실링은 PDP(플라즈마 디스플레이 패널) 산업에서 1990-2014년 사이 수억 개의 셀 봉착에 사용된 산업 검증 기술이다. Bi₂O₃(산화비스무트) 기반 무연 유리 분말("프릿")을 450°C에서 용융하면, 유리-유리 접합과 금속 와이어 매립 봉착을 동시에 달성한다.

## 병목 2: 전극 제조

<BridgeSection
  color="#6BA368"
  title="MgO 졸-겔 코팅"
  sourceTag="PDP 보호층 + 졸-겔 화학"
  cost="₩15-20K"
  diagram={<MgOCoatingDiagram />}
>

<WhyNeeded>닉시관의 수명은 **스퍼터링**(이온이 전극을 깎아내는 현상)에 의해 결정된다. MgO는 PDP에서 20년간 사용된 검증된 보호층으로, (1) 높은 2차 전자 방출 계수(γ≈0.5)로 방전 전압을 크게 낮추고, (2) 높은 결합 에너지로 이온 충돌에 강하다. SiO₂ 졸-겔이 "방어"만 한다면, MgO는 "방어 + 공격 전압 저감"을 동시에 달성.</WhyNeeded>

마그네슘 아세테이트 4수화물을 에탄올에 용해 → 옥살산으로 pH 조절 → 졸 형성 → 니켈 전극을 딥코팅 → 미니가마 400°C에서 30분 어닐링 → 결정질 MgO 박막 형성.

<InfoGrid items={[
  { label: "2차 전자 방출", value: "γ ≈ 0.5 (벌크 Ni의 5배)", color: "#6BA368" },
  { label: "어닐링 온도", value: "400°C (미니가마로 가능)", color: "#D4A853" },
  { label: "Penning 시너지", value: "Vb 30-50% ↓", color: "#7B9EB8" },
  { label: "스퍼터링 저항", value: "Ni 대비 3-5x", color: "#6BA368" },
]} />

<Caution>MgO는 수분에 민감 — Mg(OH)₂로 수화 시 γ 감소. 코팅 후 건조 환경 보관 필요. 관 봉입 전까지 데시케이터 보관 권장.</Caution>

<Evidence items={[
  "PDP 셀에서 MgO: 방전 전압 저감 + 수명 결정 요인 (20년 상용 검증)",

## 병목 3: 진공/가스 봉입

<BridgeSection
  color="#7B9EB8"

>

<WhyNeeded>전통 닉시관은 15 Torr 네온(대기압의 1/50)에서 동작하므로 진공 펌프가 필수. 그런데 파셴 법칙 V_b = f(p·d)에서 **전극 간격 d를 줄이면, 압력 p를 올려도 동일한 항복 전압**을 얻을 수 있다. 극단적으로 d를 500μm로 줄이면 200 Torr(대기압의 1/4)에서 동작 가능 → 진공 펌프 자체가 불필요해진다.</WhyNeeded>

MEMS 마이크로플라즈마 연구(2000s~)에서 대기압 근접 조건의 마이크로 방전이 활발히 연구되고 있다. 전통 닉시관의 1.5mm 갭을 500μm-1mm로 줄이면 100-500 Torr에서 동작.

<Caution>고압 방전은 발광이 "전극 전체를 감싸는" 전통 미학과 다르다. 글로우 비율 0.11-0.15는 전통의 1/3~1/2로, **"얇지만 밝은 윤곽선"**이라는 새로운 미학. 실험으로 시각적 수용 가능성 확인 필요. Ar 비율이 높으면 색상도 변할 수 있음.</Caution>

<WhyNeeded>전통 닉시관은 진공 배기 후 정밀 가스 충전한다 — 고가의 진공 펌프 + 피라니 게이지 필요. MAP(Modified Atmosphere Packaging)는 식품 산업에서 육류/채소 포장 시 공기를 N₂/CO₂로 치환하는 기법. 원리는 동일: **공기를 빼는 대신, 네온으로 밀어낸다.**</WhyNeeded>

네온 가스를 관에 관류(flow-through) 플러싱하여 공기를 물리적으로 밀어내고, O₂ 센서로 잔존 산소를 모니터링. 3회 플러싱-퍼지 사이클로 O₂ < 0.5% 달성.

<BridgeSection
  color="#7B9EB8"
  title="Penning 혼합가스 (Ne + Ar)"
  sourceTag="방전 물리 + 네온사인"
  cost="₩0 추가"
  diagram={<PenningDiagram />}
>

<WhyNeeded>순수 네온의 항복 전압(170-200V)은 감전 위험을 수반한다. Penning 혼합가스는 순수 네온 대신 소량의 아르곤(0.5-2%)을 첨가하여 **이온화 경로를 추가하고 항복 전압을 10-30% 낮추는** 물리 법칙을 활용한다. 120V 네온 램프의 표준 충전 가스로 40년+ 사용. 추가 비용 ₩0.</WhyNeeded>

Ne의 준안정 상태(metastable, 16.62 eV)가 Ar 원자(이온화 에너지 15.76 eV)와 충돌하면 Ar를 이온화할 수 있다 — 이것이 Penning 효과. 추가적인 이온+전자 쌍이 생성되므로 방전 개시가 용이해진다.

<InfoGrid items={[
