## Round 1: 인접 분야 탐색

PDP, 나노기술, MEMS에서 발견된 3개 브릿지.

<div className="rounded-xl border border-[#D4A853]/20 bg-[#D4A853]/[0.04] p-5 mb-6">

### Bridge 1: PDP 프릿 실링

| 항목 | 내용 |
|------|------|
| **출처** | PDP(플라즈마 디스플레이) 산업, 1990-2014 |
| **우회 병목** | 유리-금속 실링 (800°C → 450°C) |
| **원리** | Bi₂O₃ 기반 저융점 유리 프릿을 스크린 프린팅/디스펜싱 → 전기로 450°C 소성 → 기밀 접합 |
| **혁신성** | ★★★★☆ |
| **아마추어 비용** | ₩50,000-100,000 |
| **미검증 가정** | 0.5mm 금속 와이어의 embedded lead seal 기밀성 |
| **학술 검증** | "glass frit" AND "nixie" = 0건 |

</div>

| **학술 검증** | "cold cathode" AND "AAO" AND "glow discharge" = 0건 |

### Bridge 3: 마이크로 격벽 고압 방전

| 항목 | 내용 |

| **학술 검증** | "microplasma" AND "nixie" = 0건 |

## Round 2: 원거리 분야 탐색

건축, 자동차, 식품, 화학에서 발견된 5개 브릿지.

### Bridge G1: 부틸+폴리설파이드 상온 2중 실링

| 항목 | 내용 |
|------|------|
| **출처** | 건축 복층유리(IGU) + 자동차 전조등, 40년 산업 검증 |
| **우회 병목** | 유리-금속 실링 (800°C → **상온**) |
| **원리** | 부틸 고무(1차: 가스 차단) + 폴리설파이드(2차: 구조적 강도) |
| **혁신성** | ★★★★★ |
| **아마추어 비용** | ₩25,000 |
| **PoC 수명** | ~100일 (네온 투과 기준) |

### Bridge G2: 졸-겔 SiO₂ 기밀 보강 + 스퍼터링 방호

| 항목 | 내용 |
|------|------|
| **출처** | 화학/소재 공학 + 반도체 |
| **우회 병목** | 기밀 보강 + 전극 수명 |
| **원리** | TEOS 졸-겔로 유리질 나노코팅 → 유기 실링 위 오버코팅(투과율 10-100x 감소) + 전극 표면 스퍼터링 방호 |
| **아마추어 비용** | ₩25,000 |

### Bridge G3: MAP 가스 플러싱

| 항목 | 내용 |
|------|------|
| **출처** | 식품 산업 (Modified Atmosphere Packaging) |
| **우회 병목** | 가스 정량 충전 (진공 펌프 제거 가능) |
| **원리** | 네온 가스를 관류 플러싱하여 공기 치환 + O₂ 센서 모니터링 |
| **아마추어 비용** | ₩80,000 (가스 + 센서) |
| **제한** | 잔류 산소 0.5% → 3회 플러싱+게터로 &lt;10 ppm 달성 필요 |

### Bridge G4: 태양전지 PIB 장기 실링

| 항목 | 내용 |
|------|------|
| **출처** | 에너지/태양전지 산업 |
| **우회 병목** | 장기 수명 (부틸의 100일 → PIB의 1000일+) |
| **아마추어 비용** | ₩15,000 |

### Bridge G5: MIL-spec 기밀 커넥터

| 항목 | 내용 |
|------|------|
| **출처** | 우주/항공 산업 |
| **우회 병목** | 핀 관통 봉착 (직접 제조 → 구매) |
| **원리** | D-sub, circular 등 다양한 형태의 기밀 피드스루를 산업 부품으로 조달 |
| **아마추어 비용** | ₩20,000-50,000 |
