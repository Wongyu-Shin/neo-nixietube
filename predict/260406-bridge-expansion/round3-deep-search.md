## 발견된 유효 브릿지 (6개)

### Bridge 10: Arduino DIY 피라니 게이지 (★★★★★)

| 항목 | 내용 |
|------|------|
| **출처** | Arduino 메이커 커뮤니티 + 진공 물리 |
| **우회 병목** | 진공 측정 비용 (₩50-100K → ₩8.5K) |
| **원리** | 소형 전구 필라멘트 + Wheatstone 브리지 + ADS1115 16bit ADC + Arduino |

| **비용** | ₩8,500 (Arduino는 미니가마 PID와 공유) |
| **SWE 활용** | ★★★★★ — 캘리브레이션 코드 구현 = SWE 핵심 역량 |
| **검증** | GitHub gbochev/vacuum-gauge, Arduino Blog 2022, Hackaday 다수 |
| **핵심** | 미니가마 Arduino에 "소프트웨어 업그레이드"로 추가 |

### Bridge 12: MgO 졸-겔 코팅 (★★★★★)

| 항목 | 내용 |
|------|------|
| **출처** | PDP 산업 (1990-2014) + 졸-겔 화학 |
| **우회 병목** | 전극 수명 (스퍼터링 방호) + 방전 전압 저감 |
| **원리** | 마그네슘 아세테이트 → 에탄올 + 옥살산 → 졸-겔 딥코팅 → 400°C 어닐링 → MgO 박막 |
| **효과** | γ(2차 전자 방출) ≈ 0.5 → 방전 전압 저감 + 높은 스퍼터링 저항 |
| **비용** | ₩15-20K (MgO 전구체 + 용매) |
| **장비** | 미니가마(400°C 어닐링) — 이미 보유 |
| **검증** | PDP에서 20년간 상용 검증. 졸-겔 MgO 박막 학술 논문 다수 (MDPI, Springer) |
| **핵심** | SiO₂ 졸-겔 코팅 대비 **방전 전압 저감 효과까지 추가** |

### Bridge 13: 3D 프린트 + Ni 전해도금 전극 (★★★★)

| 항목 | 내용 |
|------|------|
| **출처** | 주얼리 전해도금 + 3D 프린팅 메이커 |
| **우회 병목** | 전극 제조 정밀도 + 복잡 형상 |
| **원리** | 레진 3D 프린트 → 도전성 페인트(구리) → Ni 전해도금(300-500μm) → 아세톤에 레진 용해 → 중공 Ni 전극 |
| **장점** | 레이저 커팅/에칭 불가능한 3D 복잡 형상 가능. 다층 전극 구조 |
| **비용** | ₩10-15K (레진 + 도전성 페인트 + 니켈 도금액) |
| **장비** | 메이커스페이스 레진 프린터 + DC 전원 |
| **검증** | Hackaday, FlashForge, Prusa Blog — 주얼리/프로토타입 분야 다수 |
| **핵심** | 기존 AAO 나노구조와 **보완적** — AAO는 표면 특성, 3D 프린트는 형상 자유도 |

### Bridge 14: Penning 혼합가스 Ne+Ar (★★★★)

| 항목 | 내용 |
|------|------|
| **출처** | 방전 물리 + 네온사인 산업 |
| **우회 병목** | 동작 전압 + 안전성 |
| **원리** | Ne 98-99.5% + Ar 0.5-2% = Penning mixture → 이온화 에너지 저감 → 항복 전압 감소 |
| **효과** | 순수 Ne 대비 항복 전압 10-30% 감소 (120V 네온 램프 표준) |
| **비용** | **₩0 추가** — 네온사인 업체에서 Ar 소량 첨가 요청 |
| **검증** | Wikipedia "Penning mixture", 120V 네온 램프에 40년+ 사용 |
| **핵심** | 파셴 시뮬레이터에 Penning 효과 추가 → 최적 Ar 비율 사전 결정 가능 |

### Bridge 15: 저융점 유리 솔더 프리폼 (★★★★★)

| 항목 | 내용 |
|------|------|
| **출처** | 광섬유 패키징 + MEMS 밀봉 |
| **우회 병목** | 프릿 페이스트 자체 조합 난이도 |
| **원리** | 링/디스크 형태의 상용 유리 솔더를 유리관 위에 배치 → 320°C 소성 → 기밀 봉착 |
| **장점** | Bi₂O₃+B₂O₃+ZnO 분말 직접 혼합 불필요. "no measurable outgassing" |
| **비용** | ₩30-50K/세트 (추정, 소량 구매 가격 확인 필요) |
| **공급** | NAMICS Corporation, OZ Optics (340-440°C), Ferro |
| **핵심** | 프릿 조합의 **분말 계량+혼합+점도 조절** 전 과정을 "기성품 구매"로 대체 |
| **미니가마 호환** | 320-440°C → 미니가마+Arduino PID로 충분히 제어 가능 |

### "Arduino 에코시스템" 최종 형태

하나의 Arduino Nano가 4가지 역할:

| 역할 | 센서/액추에이터 | 추가 비용 |
|------|-------------|---------|
| 미니가마 PID 온도 제어 | K-type 열전대 + MAX6675 + SSR | ₩21K |
| DIY 피라니 진공 게이지 | 전구 센서 + ADS1115 ADC | ₩8.5K |
| 데이터 로깅 + 디스플레이 | OLED + SD 카드 | ₩5K |
| 부스트 컨버터 안전 모니터 | 전압 분배기 + 전류 센서 | ₩3K |
| **Arduino Nano** | 공유 | ₩5K |
| **합계** | | **₩42.5K** |

이 ₩42.5K 시스템이 대체하는 상용 장비: 전기로(₩150-300K) + 피라니(₩50-100K) + 멀티미터 = **₩200-400K → 90% 절감**
