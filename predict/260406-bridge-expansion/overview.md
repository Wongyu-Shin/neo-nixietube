# Round 3: 추가 다학제 탐색 — Arduino 에코시스템 활용

**Date:** 2026-04-06
**Trigger:** 전자레인지 미니가마 발견이 보여준 것: 기존 리서치에 아직 개선 여지가 있다
**Method:** 4대 병목의 잔존 비용/기술 격차에 대해 다학제 웹서치
**Evaluation:** 6개 아이디어 탐색, 2개 유효 브릿지 발견, 4개 기각

## 발견된 새 브릿지

### Bridge 10: Arduino DIY 피라니 게이지 (★★★★★)

| 항목 | 내용 |
|------|------|
| **출처** | Arduino 메이커 커뮤니티 + 진공 물리 |
| **우회 병목** | 진공 측정 장비 비용 (₩50-100K → ₩8.5K) |
| **원리** | 텅스텐 필라멘트(또는 소형 전구) + 브리지 회로 + ADS1115 16bit ADC + Arduino |
| **측정 범위** | 0.1 Pa ~ 대기압 (= 10⁻³ Torr ~ 760 Torr) — 프로젝트 전 범위 커버 |
| **비용** | ₩8,500 (ADS1115 ₩3K + 전구 센서 ₩500 + 배선 ₩5K). Arduino는 미니가마와 공유 |
| **혁신성** | ★★★★★ — 상용 게이지 대비 95% 비용 절감 |
| **SWE 활용** | ★★★★★ — Arduino 코드 + 캘리브레이션 = SWE의 핵심 역량 |
| **학술 검증** | GitHub gbochev/vacuum-gauge, Arduino Blog (2022), Hackaday 다수 |
| **핵심 인사이트** | **미니가마 제어용 Arduino가 이미 존재 → 피라니 게이지 기능을 "소프트웨어 업그레이드"로 추가** |

**구현:**
```
[전구 센서] → [Wheatstone 브리지] → [ADS1115 16bit ADC] → [Arduino Nano]
                                                                  ↕
                                            [동일 Arduino가 미니가마 PID도 제어]
                                                                  ↕
                                            [OLED 디스플레이: 온도 + 진공도 동시 표시]
```

**비용 영향:**

## 기각된 아이디어

### 냉장고 압축기 → 진공 펌프 (기각)
- **탐색 이유:** 로터리 펌프(₩200-400K)를 자택 대체할 수 있는가?
- **결과:** 냉장고 압축기는 ~25-50 Torr까지만 도달. 프로젝트 필요 10⁻³ Torr에 크게 부족
- **판정:** ✗ 기각 — 로터리 펌프 외부 접근(네온사인 업체/대학) 유지

### 네온사인 트랜스포머 안전 회로 (기각)
- **탐색 이유:** 170-200V DC 구동 시 5mA 전류 제한 안전 회로
- **결과:** 네온사인 트랜스포머는 30mA 전류 제한으로 설계됨 — 5mA보다 6배 높음
- **판정:** ✗ 기각 — 별도 전류 제한 회로(R=33-47kΩ) + 하드웨어 퓨즈가 더 적절

### 열장벽 코팅 YSZ (과도)
- **탐색 이유:** 터빈 블레이드용 내열 코팅으로 전극 보호
- **결과:** YSZ(이트리아 안정화 지르코니아)는 1000°C+ 환경용. 닉시관 환경에 과도한 사양
- **판정:** ✗ 기각 — 졸-겔 SiO₂가 이미 적절한 수준

### MEMS 마이크로유체 가스 조절 (과도)
- **탐색 이유:** 피라니 게이지 없이 정밀 가스 충전
- **결과:** MEMS 마이크로유체 장치는 자체 제작 불가, 구매 시 ₩100K+
- **판정:** ✗ 기각 — DIY 피라니 게이지(₩8.5K)가 더 실용적

## 업데이트된 프로젝트 장비 스택

### "Arduino 에코시스템" — 하나의 MCU가 3가지 역할

| 역할 | 센서/액추에이터 | 비용 |
|------|-------------|------|
| **1. 미니가마 PID 제어** | K-type 열전대 + MAX6675 + SSR | ₩21K |
| **2. 피라니 진공 게이지** | 전구 센서 + ADS1115 16bit ADC | ₩8.5K |
| **3. 데이터 로깅** | OLED 디스플레이 + SD 카드 (선택) | ₩5K |
| **Arduino Nano** | 공유 | ₩5K |
| **합계** | | **₩39.5K** |

## 결론

**다학제 탐색 Round 3의 핵심 성과:**
1. Arduino DIY 피라니 게이지는 **"소프트웨어로 하드웨어를 대체"**하는 SWE 프로젝트의 본질적 강점을 보여줌
2. 하나의 Arduino Nano가 **가마 제어 + 진공 측정 + 데이터 로깅**을 동시 수행 → "Arduino 에코시스템"
3. 기각된 아이디어들도 유용한 정보: 냉장고 압축기의 한계(50 Torr)는 로터리 펌프가 왜 필요한지 확인해줌

**남은 외부 접근 의존성 (변동 없음):**
- 로터리 펌프: 네온사인 업체 or 대학 (10⁻³ Torr 도달에 필수, 자택 대체 불가)
- 화학 후드: 대학 화학과 (AAO 양극산화 — Phase 3에서만 필요)
