# 전자레인지용 미니가마 — 프릿 실링 450°C 적용 타당성 분석

**Date:** 2026-04-06
**Goal:** 전기로(머플로) 대신 전자레인지용 미니가마(₩40-50K)를 프릿 실링 450°C 공정에 사용할 수 있는가?
**Source:** nuboart.com 전자레인지용 미니가마 + 학술/커뮤니티 자료

## Verdict: 조건부 가능 — Arduino PID 제어 추가 시 HIGH 타당성

## 제품 사양

| 항목 | Small | Large |
|------|-------|-------|
| 가격 | ₩40,000 | ₩50,000 |
| 내부 직경 | ~70mm | ~110-124mm |
| 내부 높이 | ~45mm | ~45mm |
| 외부 높이 | ~120mm | ~110mm |
| 가열 시간 | ~5분 | 25-30분 |
| 필요 전자레인지 | 700W+ | 700W+ |
| 최고 온도 | ~900°C (1650°F) | ~900°C |

**핵심:** 최고 온도 900°C는 프릿 소성 450°C의 2배. 온도 범위는 충분.

## 프로젝트 적합성 분석

### 장점 (GO 요인)

1. **비용:** ₩50,000 vs 중고 머플로 ₩150,000-300,000 (70% 절감)
2. **접근성:** 온라인 구매 즉시 가능. 한국 내 여러 판매처 존재
3. **자택 사용:** 전자레인지만 있으면 됨. 별도 작업장 불필요
4. **챔버 크기:** Large 모델 내부 ∅110mm — 닉시관(OD 25-30mm) 충분히 수용
5. **가열 속도:** 25-30분 내 목표 온도 도달 (머플로 대비 동등)
6. **SWE 활용:** Arduino PID 제어 구현은 SWE에게 trivial

### 위험 (RISK 요인)

1. **온도 제어 없음 (CRITICAL):** 내장 온도 센서/제어 없음. 전자레인지 출력만으로 제어
2. **열전대 간섭:** 전자레인지 전자기장이 K-type 열전대에 노이즈 유발
3. **온도 구배:** 서셉터(SiC) 표면 vs 관 표면 ΔT ≈ 100°C ± 40°C (문헌 기준)
4. **분위기 제어 불가:** 바인더 번아웃(350°C)은 개방 분위기 필요 → 미니가마는 반밀폐
5. **서냉 제어 불가:** 프릿 실링 서냉(2°C/min) 달성 어려움

### 위험 완화 전략

#### Strategy A: Arduino PID 제어 (추천)

**원리:** Arduino + K-type 열전대 + SSR(Solid State Relay)로 전자레인지 전원을 PID 제어.

**구성:**
| 부품 | 사양 | 비용 |
|------|------|------|
| 전자레인지용 미니가마 (Large) | ∅110mm 챔버 | ₩50,000 |
| K-type 열전대 + MAX6675 | 0-1024°C, 디지털 출력 | ₩8,000 |
| Arduino Nano | PID 제어 | ₩5,000 |
| SSR (Solid State Relay) | 25A, 전자레인지 전원 on/off | ₩8,000 |
| 전자레인지 | 700W+, 중고 가능 | ₩0 (보유) ~ ₩30,000 (중고) |
| **합계** | | **₩71,000 ~ ₩101,000** |

**소성 프로파일 구현:**
```
Phase 1: PID → 350°C target, hold 30min (바인더 번아웃)
Phase 2: PID → 450°C target, hold 30min (프릿 융착)
Phase 3: PID → 2°C/min cooling (전자레인지 off, 자연 냉각 모니터링)
```

**열전대 간섭 해결:**
- 차폐 열전대 사용 (금속 외피 + 접지)
- 또는: 전자레인지 off 순간에만 온도 측정 (duty cycle 제어 시 off 구간 존재)
- 학술 선례: 연구자들이 차폐 K-type 열전대를 마이크로웨이브 가마에서 성공적으로 사용 (MDPI Applied Sciences 2025)

**SWE 구현 난이도:** ★☆☆☆☆ — Arduino PID 라이브러리 + MAX6675 라이브러리 = 50줄 코드

#### Strategy B: 수동 타이밍 (간이)

열전대/Arduino 없이 전자레인지 출력 조절만으로:
1. 30% 출력 × 15분 → 약 300-350°C (바인더 번아웃 추정)
2. 2분 개방 → 바인더 가스 배출
3. 60% 출력 × 10분 → 약 450°C (프릿 융착 추정)
4. 자연 냉각

**위험:** 온도 확인 불가, 과열/미달 가능성 높음. 여러 번 시도 필요.

## 결론

**Verdict: Bridge 9 — "전자레인지 미니가마 + Arduino PID"**

이것은 사실상 **새로운 브릿지 기술**이다:
- **출처:** 유리 공예 + Arduino 메이커 커뮤니티
- **우회 병목:** 전기로 접근 문제
- **원리:** SiC 서셉터 마이크로웨이브 가열 + SWE의 PID 제어 역량
- **비용:** ₩71-101K (일회성)
- **혁신성:** ★★★★ (기존 미니가마에 정밀 제어를 추가하는 조합)

**추천:**
1. 미니가마(Large, ₩50K) + Arduino PID 키트(₩21-51K) 즉시 구매
2. 온도 프로파일 검증 (빈 관으로 350°C/450°C 도달 확인)
3. 프릿 실링 시도 — 프릿 경로(경로 1)가 자택에서도 가능해짐

**프로젝트 영향 (대규모 방향 전환):**
- 경로 1(프릿)의 "도자기 공방 가마 접근" 필요성 **제거**
- 어닐링(유리 응력 제거, 400-550°C)에도 동일 장비 사용 가능
- 경로 1도 경로 2처럼 **자택에서 시작 가능**
- 유일하게 남는 외부 접근 필요: 로터리 펌프 (네온사인 업체/대학)
- **결과:** 프로젝트의 6개 열처리 공정 중 전기로 의존 공정(프릿 소성, 어닐링) 2개가 자택으로 전환
- 나머지 4개(베이킹-히트건, 팁오프-토치, 게터활성화-RF, 진공배기-펌프)는 원래 전기로가 아닌 별도 장비
