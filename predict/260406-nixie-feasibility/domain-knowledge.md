# Domain Knowledge — Nixie Tube Manufacturing

## 1. 닉시관(Nixie Tube) 기본 구조

닉시관은 냉음극 방전관(Cold Cathode Discharge Tube)의 일종.

| 구성 요소 | 역할 | 재료/사양 |
|-----------|------|----------|
| 외피(Envelope) | 진공/가스 봉입 용기 | 보로실리케이트 유리(Pyrex 등), 소다라임 유리 |
| 양극(Anode) | 공통 전극, 금속 메쉬 | 니켈 메쉬, 스테인리스 스틸 메쉬 |
| 음극(Cathode) | 숫자/기호 형상 전극 (0-9 등) | 니켈 판, 냉간압연 금속 박 |
| 핀(Pins) | 외부 연결 단자 | 듀멧 와이어(Fe-Ni-Cu 복합), 코바르(Kovar) 합금 |
| 봉입 가스(Fill Gas) | 글로우 방전 매질 | 네온(Ne) 주체, 소량 아르곤(Ar)/수은(Hg) 첨가 |
| 게터(Getter) | 잔류 가스 흡착 | 바륨(Ba), 지르코늄(Zr) |

## 2. 제조 공정 개요

### 2.1 전극 제조
- 니켈 판재를 CNC/레이저/화학 에칭으로 숫자 형상 커팅
- 각 음극 숫자를 간격 유지하여 적층 (1~2mm 간격)
- 양극 메쉬를 최전면에 배치
- 접점/리드선 용접 (스폿 용접 또는 납땜)

### 2.2 유리 가공
- 관형(tubular) 또는 평판형(flat-front) 외피 제작
- 유리-금속 봉착(Glass-to-Metal Seal): 핀을 유리를 통과시켜 기밀 봉착
- 보로실리케이트 유리: 열팽창계수 ~3.3×10⁻⁶/K
- 코바르 합금: 열팽창계수 ~5.0×10⁻⁶/K (보로실리케이트와 호환)
- 듀멧 와이어: 소다라임 유리용 봉착 재료

### 2.3 진공/가스 공정
- 진공 배기: ~10⁻⁵ Torr 이상의 진공도 필요
- 베이킹(Bakeout): 250-350°C에서 수시간 탈가스
- 가스 충전: 네온 가스 10-30 Torr 봉입
- 봉지(Tip-off): 배기관을 가열하여 밀봉
- 게터 활성화: 고주파 가열로 게터 활성화

### 2.4 에이징/테스트
- 정격 전압 인가하여 방전 안정화 (수시간~수일)
- 전 음극 순차 점등 확인
- 스퍼터링(음극 물질 비산) 최소화 검증

## 3. 핵심 물리학

### 3.1 글로우 방전 (Glow Discharge)
- 파셴 법칙(Paschen's Law): V_b = f(p·d)
  - V_b: 항복 전압
  - p: 가스 압력
  - d: 전극 간 거리
- 네온 가스 최소 항복 전압: ~180V (p·d ≈ 0.5-1.5 Torr·cm)
- 일반 동작 전압: 170-200V DC (양극-음극 간)

### 3.2 네온 발광
- 네온 오렌지-레드 발광: 585-640nm (주 파장 585.2nm, 640.2nm)
- 발광 메커니즘: 전자 충돌 여기 → 탈여기 시 광자 방출
- 소량 Hg 첨가 시: 스퍼터링 감소, 수명 증가 (독성 주의)

### 3.3 진공 기술
- 거친 진공(Rough vacuum): ~10⁻³ Torr (로터리 펌프)
- 고진공(High vacuum): ~10⁻⁶ Torr (터보분자펌프/확산펌프)
- 닉시관 요구: 배기 시 10⁻⁵ Torr → 네온 충전 후 10-30 Torr
- 리크 레이트 허용치: <10⁻⁸ Torr·L/s (유리-금속 봉착부)

## 4. 2026년 기준 관련 기술 발전

### 4.1 제조 기술
- **데스크톱 레이저 커터**: CO2/파이버 레이저, 금속 박 커팅 가능 (xTool, Glowforge 등)
- **데스크톱 CNC**: 알루미늄/니켈 박 가공 가능 (Nomad, PocketNC)
- **화학 에칭**: FeCl3 기반 DIY PCB 에칭 기법 → 니켈 박에 적용 가능
- **3D 프린팅**: 유리 직접 프린팅은 아직 산업급, SLA/SLS로 몰드 제작 가능
- **데스크톱 스폿 용접기**: 18650 배터리 용 스폿 용접기 ($50-200)

### 4.2 진공 기술
- **중고 진공 펌프**: 로터리 펌프 (eBay/중고 장터, $200-500)
- **터보분자펌프**: 중고 가능하나 고가 ($1000+), 컨트롤러 별도
- **DIY 진공 시스템**: 냉매 트랩 + 로터리 펌프 조합으로 10⁻⁴ Torr 도달 가능
- **헬륨 리크 디텍터**: 산업용 (접근 어려움), 대안: 방전 색상 검사, 압력 유지 테스트

### 4.3 유리 가공
- **램프워킹(Lampworking)**: 보로실리케이트 유리 토치 가공 (아마추어 접근 가능)
- **유리-금속 봉착**: Housekeeper seal 기법 (얇은 금속 → 유리 직접 용착)
- **상업용 유리관**: 각종 직경의 파이렉스 관 온라인 구매 가능

### 4.4 가스/재료
- **네온 가스**: 산업용 고순도 네온 (99.999%) 구매 가능하나 고가
- **대안**: 소등 가스 혼합물 (네온-아르곤), 네온사인 업체에서 소량 구매
- **게터**: 진공관 DIY 커뮤니티에서 소량 구매 가능
- **코바르/듀멧**: 전자부품 전문 업체에서 소량 구매 (국내: 전자부품 전문몰)
