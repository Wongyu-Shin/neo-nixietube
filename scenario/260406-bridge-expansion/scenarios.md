## Scenario 1: MgO 졸-겔 코팅 → 프릿 소성 열 호환성

**상황:** MgO 졸-겔로 니켈 전극을 코팅(400°C 어닐링) → 이후 프릿 소성(450°C)에서 MgO 막이 생존하는가?

**분석:**
- MgO 용융점: 2,852°C → 450°C에서 안전 (안전 마진 2400°C)
- PDP에서 MgO는 500°C+ 프로세스 후에도 기능 유지 (문헌 확인됨)
- 프릿의 Bi₂O₃ 용융물이 MgO 막과 반응할 가능성: 낮음 (MgO는 화학적 안정)
- **판정: SAFE** — MgO 코팅 후 프릿 소성 가능

**추천 공정 순서:**
1. 전극 제작 (레이저 커팅/에칭)
2. MgO 졸-겔 딥코팅 + 미니가마 400°C 어닐링
3. 프릿 페이스트 도포 + 듀멧 배치
4. 미니가마 350°C(바인더) → 450°C(융착) — MgO는 이 온도에서 안전

## Scenario 2: Penning 혼합가스(Ne+Ar) → 파셴 시뮬레이터 영향

**상황:** 기존 시뮬레이터는 순수 Ne 기반(MAPE 1.79%). Ne+0.5-2%Ar Penning 혼합 시 시뮬레이터 정확도는?

**분석:**
- Penning 효과는 표준 파셴 공식에 포함되지 않음 (추가 이온화 경로)
- Ar의 준안정 상태(metastable) 에너지 = 11.55 eV, Ne 이온화 에너지 = 21.56 eV → 직접 Penning 이온화 불가
- 그러나 Ar* + Ne → Ar + Ne⁺ + e⁻ (에너지 전달)는 비효율적
- **실제 Penning 효과:** Ne*가 Ar를 이온화 (Ne 준안정 16.62 eV > Ar 이온화 15.76 eV) → **이 방향이 맞음**
- 시뮬레이터에 Penning cross-section 항 추가 필요

**판정:** 시뮬레이터 업데이트 권장. 그러나 **실험적 확인이 가장 빠름** — 순수 Ne와 Ne+1%Ar를 같은 관에서 비교.

**리스크:** Ar 비율이 너무 높으면 (>2%) 글로우 색상이 변함 (오렌지 → 보라/핑크). 닉시관의 "따뜻한 오렌지" 미학에 영향.

## Scenario 3: 유리 솔더 프리폼 → 듀멧 와이어 매립 호환성

**상황:** 상용 유리 솔더 프리폼(링 형태, 320-440°C)에 듀멧 와이어를 매립할 수 있는가?

**분석:**
- 유리 솔더 프리폼은 **미리 성형된 형태**로 공급
- 듀멧 와이어는 프리폼 위에/아래에 배치 → 소성 시 용융된 프리폼이 와이어를 감싸야 함
- **핵심 질문:** 프리폼의 점도(소성 시)가 와이어 주변을 충분히 적시는가(wetting)?
- OZ Optics 제품은 "광섬유 금속 기밀 봉착"용 → 금속 와이어 매립에 설계된 제품
- **판정: LIKELY COMPATIBLE** — 광섬유 패키징은 금속-유리 기밀 접합 목적이므로, 듀멧 와이어에도 적용 가능

**엣지케이스:**
- 프리폼 링의 내경이 유리관 OD(25mm)와 맞아야 함 → 맞춤 주문 필요할 수 있음
- 표준 프리폼은 광섬유 크기(∅125μm) → 닉시관 크기(OD 25mm)는 **수백 배 큰** 스케일
- **해결:** OZ Optics의 맞춤 형상 서비스, 또는 NAMICS의 프리폼 페이스트(비성형) 사용

## Scenario 4: 3D 프린트 전극 → 진공 환경 호환성

**상황:** 3D 프린트 레진 코어를 아세톤으로 용해한 후, 잔류 유기물이 진공 환경에서 문제를 일으키는가?

**분석:**
- 아세톤 용해 후 잔류물: 수 ppm 수준의 유기 잔류물
- 진공 배기(10⁻³ Torr) + 베이킹(200°C)으로 대부분 제거 가능
- 그러나 나노스케일 pore에 갇힌 잔류물은 장기적으로 outgassing → 글로우 오염
- **판정: MEDIUM RISK** — 충분한 세척(초음파 + IPA + 베이킹) 필요

**완화 전략:**
1. 용해 후 IPA 초음파 세척 30분
2. 히트건 200°C 베이킹 2시간 (outgassing 가속)
3. 그래도 순수 니켈 포일 전극보다 불순물 높을 수 있음 → **PoC에는 기존 레이저/에칭 전극을 권장, 3D 프린트는 Phase 4 이후**

## Scenario 5: Arduino 에코시스템 — 동시 실행 충돌

**상황:** 하나의 Arduino Nano로 미니가마 PID + 피라니 게이지 + 데이터 로깅을 동시에 실행할 때, 처리 시간 충돌이 발생하는가?

**분석:**
- Arduino Nano: 16MHz ATmega328P, 32KB flash, 2KB SRAM
- PID 루프: ~10ms 주기 (100Hz) — MAX6675 읽기 250ms → 4Hz가 현실적
- 피라니 ADC: ADS1115 읽기 ~10ms
- OLED 업데이트: ~50ms
- SD 카드 쓰기: ~20ms

**시간 예산 (1초 주기):**
| 작업 | 시간 | 빈도 |
|------|------|------|
| MAX6675 온도 읽기 | 250ms | 4x/s |
| PID 계산 + SSR 출력 | 1ms | 4x/s |
| ADS1115 진공 읽기 | 10ms | 2x/s |
| OLED 디스플레이 | 50ms | 1x/s |
| SD 카드 로깅 | 20ms | 1x/s |
| **총** | **~340ms/s** | 여유 660ms |

**판정: NO CONFLICT** — ATmega328P로 모든 작업 동시 실행 가능. 여유 66%.

**엣지케이스:** MAX6675은 전자레인지 동작 중 전자기 간섭 가능 → **SSR off 구간(PID duty cycle의 off 시간)에만 온도 읽기** 권장.

## Scenario 7: 최악의 경우 — 모든 새 브릿지 실패

**상황:** MgO 코팅 실패, 프리폼 구매 불가, Penning 효과 미미, 3D 프린트 전극 오염.

**분석:** 
- Round 1-2의 기존 브릿지는 모두 유지됨
- 프릿 자체 조합(Bi₂O₃+B₂O₃+ZnO)은 여전히 유효
- 졸-겔 SiO₂ 코팅은 여전히 유효
- 순수 Ne는 여전히 유효
- **Round 3 브릿지는 "추가 최적화"이지 "필수 전제조건"이 아님**

**판정:** Round 3 실패 시 프로젝트는 Round 2 상태로 복귀. 비용 ₩550-650K. 기존 분석의 타당성은 유지.

## Scenario 8: MgO + Penning 시너지

**상황:** MgO 코팅(γ ≈ 0.5) + Penning 혼합가스(Ne+Ar)를 동시 적용.

**분석:**
- MgO의 높은 γ → 더 적은 이온 충돌로도 방전 유지 가능
- Penning 효과 → 이온화 에너지 추가 저감
- **시너지:** 항복 전압이 순수 Ne + 벌크 Ni 대비 **30-50% 저감** 가능 (추정)
- 150V 수준까지 낮아지면 → 부스트 컨버터 요구 전압 감소 → 안전성 향상
- 낮은 전압 = 이온 에너지 감소 = **스퍼터링 감소** → 수명 추가 연장

**판정: HIGH SYNERGY** — 두 브릿지의 조합이 각각보다 큰 효과. PoC에서 반드시 테스트.

## Scenario 9: 유리 솔더 프리폼 소량 구매 실패

**상황:** NAMICS/OZ Optics에서 소량(1-10개) 구매가 불가능.

**대안:**
1. **자체 프릿 조합 (기존 방법):** Bi₂O₃+B₂O₃+ZnO 분말 혼합. ₩90K. 난이도 중.
2. **Bullseye 유리 공예 프릿:** 유리 공예 프릿은 소량 구매 가능하지만, 조성(소다라임 기반)이 다름. 융점 700°C+ → 미니가마 용량 내이지만 프릿 실링 목적의 저융점이 아님.
3. **중국 AliExpress:** 저융점 유리 프릿 분말 소량 판매 존재 가능성 → 검색 필요
4. **YEKGLASS (한국?):** 웹사이트에서 무연 저융점 프릿 분말+페이스트 판매. 소량 구매 문의 가능.

**판정:** 프리폼이 불가 시 자체 조합으로 충분히 진행 가능. 프리폼은 "편의" 개선이지 "필수"가 아님.

## Scenario 10: 안전 시나리오 — 전자레인지 + Arduino + 고전압

**상황:** 전자레인지(220V AC) + SSR + Arduino + 부스트 컨버터(200V DC)가 같은 작업대에 있을 때.

**위험 매트릭스:**
| 위험 | 심각도 | 확률 | 완화 |
|------|--------|------|------|
| 전자레인지 마이크로파 누출 | 높음 | 낮음 | 열전대 관통 구멍을 파장 이하(< 12cm)로 유지 |
| SSR 고장 → 전자레인지 과열 | 높음 | 중 | Arduino watchdog + 기계식 타이머 백업 |
| 고전압 접촉 | CRITICAL | 중 | one-hand rule, 절연 장갑, 블리드 저항 |
| Arduino 리셋 → PID 실패 | 중 | 낮음 | 전자레인지 off로 안전 실패 (fail-safe) |

**추가 안전 조치:**
1. 전자레인지와 고전압 회로를 **물리적으로 분리** (최소 1m)
2. SSR에 **기계식 타이머**(30분) 추가 → Arduino 실패 시에도 전자레인지 자동 off
3. 부스트 컨버터에 **5mA 퓨즈** 하드웨어 수준에서 설치
4. **비상 정지 버튼** — 모든 전원 일괄 차단

## 핵심 시사점 요약

1. **MgO + Penning 시너지가 가장 높은 ROI** — 비용 ₩15K 추가로 방전 전압 30-50% 저감 + 수명 연장
2. **유리 솔더 프리폼은 "편의"** — 구매 가능하면 좋지만, 자체 조합도 유효
3. **3D 프린트 전극은 Phase 4 이후** — 잔류물 문제로 PoC에서는 기존 방법 권장
4. **Arduino 에코시스템은 처리 여유 충분** — 4가지 작업 동시 실행 가능 (여유 66%)
5. **Round 3 브릿지 실패해도 프로젝트 무영향** — 기존 Round 1-2 브릿지로 충분
6. **안전은 물리적 분리 + 기계적 백업** 원칙
