# Edge Cases & Failure Modes — Nixie Tube Manufacturing

## CRITICAL (비협상 — 생명 위협)

### EC-1: 캐패시터 잔류 전하 감전
- **Phase:** 테스트/디버깅
- **Trigger:** 전원 OFF 후 출력 캐패시터(10μF/250V) 접촉
- **Prevention:** 1MΩ 블리드 저항 장착, 전원 OFF 후 10초 대기, 멀티미터 확인
- **Checklist:** [ ] 블리드 저항 장착 [ ] 절연장갑 착용 [ ] 전압 확인 후 작업

### EC-2: 유리 내파 파편 비산
- **Phase:** 진공 배기
- **Trigger:** 진공 유리관에 외부 충격
- **Prevention:** 보안경/면갑 상시 착용, 30cm 반경 정리, 메쉬 슬리브
- **Checklist:** [ ] 보안경 착용 [ ] 작업대 정리 [ ] 유리관 래핑

### EC-3: 토치 역화(Flashback)
- **Phase:** 유리 가공
- **Trigger:** 가스 유량 부족 상태에서 화염 역류
- **Prevention:** 역화 방지기 장착 (₩20,000-40,000), 점화 순서 준수
- **Checklist:** [ ] 역화 방지기 확인 [ ] 소화기 비치 [ ] 환기 확보

## HIGH (반드시 대비)

### EC-4: 니켈 포일 순도 위장
- **Given:** 세운상가에서 "니켈 포일" 구매
- **When:** 실제로는 니켈 도금 강판
- **Then:** 스퍼터링 가속 → 수십 시간 만에 유리 흑화
- **Mitigation:** 전해 니켈(99.5%+) 전문 업체 구매. 자석 테스트(순수 Ni는 약자성)

### EC-5: 유리-금속 CTE 불일치
- **Given:** 소다라임 유리에 코바르 와이어 봉착
- **When:** CTE 차이 4.0×10⁻⁶/K
- **Then:** 냉각 시 방사형 크랙 → 수일 내 파손
- **Mitigation:** 매칭 규칙: 소다라임↔듀멧, 보로실리케이트↔코바르/텅스텐

### EC-6: 팁오프 불완전 → 핀홀 리크
- **Given:** 배기관 토치 가열
- **When:** 유리 완전 붕괴 전 토치 제거
- **Then:** 핀홀 리크 → 수시간~수일에 걸쳐 공기 유입
- **Mitigation:** 완전 붕괴 확인 + 팁오프 직후 1시간 hold test

### EC-7: 피라니 게이지 보정 오류
- **Given:** 공기 교정 피라니 게이지로 네온 압력 측정
- **When:** 네온의 열전도율이 공기와 다름
- **Then:** 표시 15 Torr → 실제 ~13 Torr → 파셴 범위 이탈
- **Mitigation:** 가스별 보정 계수 적용 또는 가스 종류 무관 게이지(캐패시턴스 마노미터) 사용

### EC-8: 펌프 오일 backstreaming
- **Given:** 중고 로터리 펌프, 오일 미교환
- **When:** 탄화수소 증기 역류
- **Then:** 관 내부 오염 → 이상 발색 (녹색/파란색)
- **Mitigation:** 오일 교환 필수. 포어라인 트랩(분자체) 추가

### EC-9: 인접 음극 크로스토크 방전
- **Given:** 음극 간격 1.0mm, 네온 15 Torr
- **When:** p·d 값이 인접 음극 간에서도 최적 범위
- **Then:** 의도치 않은 음극에서 부분 방전 → 숫자 겹침
- **Mitigation:** 간격 2.0mm+, 마이카 스페이서, anti-cathode bias

### EC-10: 배기 중 전극 조립체 이탈
- **Given:** 진공 배기 시 압력차
- **When:** 전극-핀 접합이 약함
- **Then:** 전극이 밀려나 정렬 파괴
- **Mitigation:** 마이카 고정 + 스폿 용접 + 배기 반대쪽 지지점

### EC-11: 게터 활성화 시 봉착부 연화
- **Given:** RF 유도 가열 코일이 봉착부에 근접
- **When:** RF 에너지가 인접 핀도 가열
- **Then:** 봉착 유리 연화 → 기밀 파괴 → 관 폐기
- **Mitigation:** 게터를 봉착부에서 최대 이격. 코일 직경 최소화. 또는 게터 미사용

### EC-21: 스파크 코일 리크 테스트 위양성
